## 📌 About the project

**Fern & Fog** is my pet project — a minimal, audio-first platform.  
The idea is simple: open the page, press play, and exist in a quiet space. Later — upload your own sounds.

**Current status:** Alpha / active development

---

## 🛠 Tech stack (current)

| Layer | Technology |
|-------|-------------|
| Frontend | HTML5, CSS3, Vanilla JS |
| Hosting + CDN | [Netlify](https://netlify.com) (free tier, 100GB traffic/month) |
| Code storage | GitHub (private repo for source code) |
| Analytics | [Umami](https://umami.is) (privacy-friendly) |

**Migrated from:** Webflow → custom static site (more control, free audio player on homepage)

## 🧩 Project structure (explained)

- **This public repository** → documentation, roadmap, design notes, announcements  
- **Private repository** → actual source code (`index.html`, `style.css`, `src/`, etc.)  
- **Live site** → auto-deployed from private repo via Netlify

The source code is private while the project is in early stage. I may open-source it later.
