# 🌿 Fern & Fog — community-driven sanctuary of sound

## About the project
**Fern & Fog** is a calm online space for people seeking peace and escape from daily noise.  
Users can listen to ambient music and nature sounds, share their own recordings, and support each other.

## Current status
The project is under active development. Currently implemented:

- ✅ Home page with a background illustration and a vertical menu
- ✅ Profile page with navigation and an "Add sound" button
